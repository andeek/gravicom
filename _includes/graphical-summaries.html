Graphical Summaries

The graphical summaries module provides several common graphical displays of variables in the dataset. First, the user selects between two types of plots, either one variable, or two variables. This choice determines the plot types presented to the user. The user then selects a plot type, and either a single variable, or two variables from the dataset. The plot visible on the right will automatically update when these values are selected. The graphical displays available are:

One Variable
  • Histogram
  • Boxplot
Two Variables
  • Scatterplot
  • Line Chart
  • Boxplot
  • Bar Chart
  • Pareto Chart

If there are options corresponding to the plot type selected, they will be displayed beneath the plot in the results panel. Currently, the plots with options are bar charts, pareto charts, and histograms. For bar charts and pareto charts, the user has the ability to select what statistic is displayed on the Y axis. Options include count, sum, mean, and median. For the histograms, users can select the bin width corresponding to the width of the bar.
